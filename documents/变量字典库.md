# 国家统计局泰兴调查队公文处理系统 V3.0 - 变量字典库

> **说明**：本文件作为全系统（前后端、数据库、AI 引擎）变量命名与枚举值的唯一受信任来源。所有开发变更必须同步更新此表，严禁在代码中硬编码逻辑字符串。

---

## 1. 业务状态枚举 (Business Enums)

### 1.1 公文状态 (`DocumentStatus`)
| 变量名 | 数据库存储值 | 前端展示名 | 描述说明 |
| :--- | :--- | :--- | :--- |
| `DRAFTING` | `DRAFTING` | 起草中 | 允许编辑、润色、手动排版预览 |
| `SUBMITTED` | `SUBMITTED` | 待审批 | 锁定编辑，进入科长待办 |
| `APPROVED` | `APPROVED` | 已通过 | 终态，生成 SIP 指纹，触发物理排版 |
| `REJECTED` | `REJECTED` | 已驳回 | 锁定编辑，需由起草人触发 `revise` 回退 |

### 1.2 知识库层级 (`KBTier`)
| 变量名 | 数据库存储值 | 描述说明 |
| :--- | :--- | :--- |
| `BASE` | `BASE` | 全局基础库（仅管理员可写） |
| `DEPT` | `DEPT` | 科室共享库（仅科长可写） |
| `PERSONAL` | `PERSONAL` | 个人沙箱库（仅属主可读写） |

### 1.3 数据安全等级 (`DataSecurityLevel`)
| 变量名 | 数据库存储值 | 前端标记 | 过滤策略 |
| :--- | :--- | :--- | :--- |
| `CORE` | `CORE` | 🔒 核心 | 禁止 RAG 召回，仅本地校验 |
| `IMPORTANT` | `IMPORTANT` | ⚠️ 重要 | 允许 RAG，前端强制高频水印 |
| `GENERAL` | `GENERAL` | 公开 | 正常流转与召回 |

## 2. 数据库结构标识 (Database Entities)

### 2.1 核心表名 (`Table Names`)
- `system_users`: 用户表
- `departments`: 科室组织表
- `users_sessions`: 会话与单设备登录控制表
- `documents`: 公文主表
- `document_types`: 公文文种配置表
- `exemplar_documents`: 参考范文库
- `document_snapshots`: 历史快照（后悔药）表
- `document_approval_logs`: 审批存证与归档日志
- `knowledge_base_hierarchy`: 知识库逻辑目录树
- `knowledge_chunks`: 向量切片与特征存储
- `knowledge_physical_files`: 物理文件去重表
- `async_tasks`: 异步任务持久化记录
- `nbs_workflow_audit`: 全生命周期合规审计流
- `user_notifications`: 系统通知持久化表
- `system_config`: 动态系统参数表

## 3. 异步任务与实时通信 (Async & SSE)

### 3.1 任务类型 (`TaskType`)
| 变量名 | 存储值 | 描述 |
| :--- | :--- | :--- |
| `POLISH` | `POLISH` | AI 语义润色任务 |
| `FORMAT` | `FORMAT` | GB/T 9704-2023 排版任务 |
| `PARSE` | `PARSE` | 知识库文件解析任务 |

### 3.2 SSE 事件标识 (`Event Types`)
- `task.progress`: 任务进度更新 (0-100)
- `task.completed`: 任务成功完成
- `task.failed`: 任务执行失败
- `notification.rejected`: 公文被驳回通知
- `notification.approved`: 公文审批通过通知
- `notification.lock_reclaimed`: 编辑锁被强制回收通知
